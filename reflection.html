心得一：學習 HTML 的體驗

剛開始接觸 HTML 時，覺得語法簡單但邏輯很重要。透過實作網頁，我學會了如何組織資訊、建立清楚的內容結構。

心得二：CSS 美化技巧

CSS 讓靜態的網頁變得生動有趣。從顏色配置、排版到 hover 效果，讓我了解美感與實用要並重。

心得三：JavaScript 初探

JavaScript 幫助我讓網站「活」了起來。即使只會簡單功能如按鈕互動，也能大大提升使用者體驗。

心得四：作品整合與發佈

透過 GitHub Pages 發佈我的網站，是一個非常有成就感的體驗。從本機端到雲端，每一步都很有學習價值。
